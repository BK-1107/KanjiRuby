# KanjiRuby - 日语汉字注音助手

这是一个能够自动为网页上的日语汉字添加平假名注音的浏览器开发工具。它由 Python 后端（提供分词分析）和 Chrome 插件（负责前端显示）组成。

## 功能特性 (Features)

*   **自动注音**: 选中网页上的日语文本，自动识别汉字并标注假名。
*   **沉浸式翻译**: 选中文字后，在原文下方直接显示中文翻译，采用“双层注音”排版，不破坏原网页结构。
*   **双击 Ctrl 快捷开关**: 支持连按两次 `Ctrl` 键快速开启/关闭翻译模式，并带有精致的毛玻璃效果提示弹窗 (Toast)。
*   **HTTPS 全兼容**: 通过 Background Proxy 机制，完美支持在加密网页（如 Syosetu, Yahoo, NHK 等）上运行，不再受混合内容限制。
*   **全网页一键注音**: 点击插件面板按钮，可对当前页面所有文本节点进行快速扫描并标注假名。
*   **解耦设计**: 注音功能与翻译功能可独立开关，互不干扰。
*   **一键还原**: 随时清除页面上的所有标注和翻译，恢复原始网页排版，且不丢失滚动位置。
*   **排版保护**: 完美保留原始文本的换行和段落结构，支持跨段落选取而不会导致文本合并。

## 安装与运行 (Setup)

本项目包含两部分，都需要运行才能正常工作。

### 1. 启动后端服务 (Backend)

确保您的环境已安装 Python 3.8+。

**安装依赖:**
```bash
pip install fastapi uvicorn fugashi unidic-lite
```

**启动服务:**
直接双击项目根目录下的 **`run_backend.bat`** 即可一键启动。
*优化提示：该脚本支持 `Ctrl+C` 秒退，无需点击确认。*

### 2. 加载浏览器插件 (Extension)

1.  打开 Chrome 浏览器，在地址栏输入 `chrome://extensions/`。
2.  开启右上角的 **"开发者模式" (Developer mode)** 开关。
3.  点击左上角的 **"加载已解压的扩展程序" (Load unpacked)**。
4.  选择本项目文件夹下的 **`extension`** 目录。

## 使用方法 (Usage Tips)

1.  **快速切换**: 在设置中开启 **Double-Tap Shortcut** 后，连按两下键盘 `Ctrl` 键即可快速切换翻译模式。
2.  **即时翻译**: 在切换翻译模式到 ON 的瞬间，如果当前正选中有文本，插件会立即请求翻译。
3.  **注音优先**: 默认模式下仅注音。如果想在阅读时偶尔看翻译，推荐使用 `Ctrl` 双击快捷键。

## 项目结构

```
.
├── backend/            # Python 后端核心逻辑
├── extension/          # Chrome 插件源码
│   ├── background.js   # API 代理中转 (解决 HTTPS 安全限制)
│   ├── content.js      # 前端逻辑 (DOM 操作与交互)
│   ├── popup.html      # 插件控制面板
│   └── manifest.json   # 插件配置 (V3)
├── main.py             # FastAPI 服务入口
├── run_backend.bat     # 优化过的后端启动脚本
└── README.md           # 本文档
```
