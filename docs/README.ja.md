# KanjiRuby - 日本語漢字ふりがなアシスタント

日本語 | [简体中文](../README.md)

KanjiRuby は、日本語の読解学習を支援する Chrome 拡張機能です。Web ページ上の日本語の漢字にふりがなを付け、必要に応じて選択した日本語テキストを中国語へ自然に翻訳できます。小説、ニュース、ブログ、学習教材を読むときの負担を軽くします。

> 現在のバージョンはローカルの Python バックエンドサービスに依存しています。使用する前に、先にバックエンドを起動してから Chrome 拡張機能を読み込んでください。

## 表示例

### 漢字のふりがな表示

Web ページ上の日本語テキストを選択すると、KanjiRuby はそのテキストをローカルバックエンドへ送信して解析します。バックエンドは `fugashi + unidic-lite` を使って形態素解析と読みの取得を行い、拡張機能がページ上に Ruby 形式でふりがなを表示します。

## 主な機能

- **漢字の自動ふりがな表示**：Web ページ内の日本語漢字を識別し、Ruby 形式でひらがなの読みを表示します。
- **選択テキストの翻訳**：選択した日本語テキストの下に中国語訳を表示します。
- **ページ全体の一括ふりがな表示**：拡張機能パネルから現在のページをスキャンし、テキストへまとめてふりがなを付けます。
- **Ctrl ダブルタップ切り替え**：`Ctrl` キーを 2 回押して、翻訳モードのオン・オフを素早く切り替えられます。
- **TTS 読み上げ**：日本語テキストを音声で読み上げ、リスニングと読解を補助します。
- **一括クリア**：ページ上のふりがなと翻訳を削除し、元の表示に戻します。
- **レイアウト保護**：元ページの段落、改行、テキスト構造をできるだけ保ち、読みやすさを損なわないようにします。

## プロジェクト構成

KanjiRuby は次の 2 つの部分で構成されています。

- **Chrome 拡張機能**：ページ操作、テキスト選択、ふりがな表示、翻訳表示、コントロールパネルを担当します。
- **Python バックエンド**：日本語の形態素解析、読みの取得、翻訳リクエストを担当します。

現在のバージョンはローカルバックエンドサービスに依存しています。Chrome 拡張機能は、選択された日本語テキストを `http://127.0.0.1:18000/analyze` に送信し、FastAPI バックエンドから分詞結果、ふりがな、必要に応じた中国語訳を受け取ります。

拡張機能だけをインストールしてバックエンドを起動していない場合、ふりがなと翻訳のリクエストは完了できません。

## インストールと起動

このプロジェクトを使うには、バックエンドサービスとブラウザ拡張機能の両方を準備する必要があります。

### 1. バックエンドサービスを起動する

Python 3.8 以上がインストールされていることを確認してください。

依存関係をインストールします。

```bash
pip install -r requirements.txt
```

手動でインストールする場合：

```bash
pip install fastapi uvicorn fugashi unidic-lite pydantic httpx deep-translator
```

サービスを起動します。

```text
run_backend.bat
```

Windows では、プロジェクトルートの `run_backend.bat` を直接ダブルクリックして起動することもできます。サービスはデフォルトで次のアドレスで動作します。

```text
http://127.0.0.1:18000
```

### 2. Chrome 拡張機能を読み込む

1. Chrome を開き、アドレスバーに `chrome://extensions/` と入力します。
2. 右上の「デベロッパー モード」をオンにします。
3. 「パッケージ化されていない拡張機能を読み込む」をクリックします。
4. このプロジェクト内の `extension` ディレクトリを選択します。

読み込みが完了すると、Chrome のツールバーに KanjiRuby の拡張機能アイコンが表示されます。

## 使い方

### ふりがな

Web ページ上の日本語テキストを選択し、拡張機能パネルからふりがなを実行します。KanjiRuby はテキスト内の漢字を識別し、ページ上にひらがなの読みを追加します。

### 翻訳

翻訳モードをオンにした状態で日本語テキストを選択すると、中国語訳を確認できます。翻訳機能は現在、`deep-translator` 経由で Google Translate を使用しています。

### ショートカット切り替え

拡張機能の設定で Double-Tap Shortcut を有効にすると、`Ctrl` キーを 2 回押して翻訳モードを素早く切り替えられます。切り替え時にはページ上に通知が表示されます。
